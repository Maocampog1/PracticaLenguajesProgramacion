Creación de Seres Vivos
Integrantes del equipo:
• Maria Alejandra Ocampo

Detalles de implementación:
• Lenguaje: Java
• Versión del compilador: Java 17
• IDE utilizado: IntelliJ

Notas: 
• Utilizamos herencia e interfaces para organizar y compartir comportamientos comunes entre diferentes tipos de seres vivos.
• Usamos la separación de paquetes y modificadores de acceso para controlar el acceso a características de criaturas del Cielo desde criaturas de la Tierra.
• Implementa el patrón de diseño Factory utilizando la interfaz Creador para crear dinámicamente instancias de seres vivos.
